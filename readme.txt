Use AccessQontrol

== Description ==

EasyBan is a discontinued thing. THe important features have been moved into AccessQontrol, another plugin of mine.
[http://meandmymac.net/plugins/accessqontrol/](http://meandmymac.net/plugins/accessqontrol/)

**Features**

* Easy installation
* Blocks persons (or domains) with a hostname or IP address from your site
* Shows a nice template

== Installation ==

= Quick start =

1. Unzip the file and upload it to your /wp-content/plugins/ folder.
1. Activate from the plugins page and manage bans from the manage > banned page.

[Additional information](http://meandmymac.net/plugins/easyban/#install)
